# كيفية إضافة خدمة جديدة للنظام

دليل شامل خطوة بخطوة بالعربي لإضافة أي خدمة جديدة لنظام الـ API والـ Hooks في مشروعك.

## 📋 فهم البنية الحالية

قبل ما نبدأ، لازم نفهم البنية بتاعت المشروع:

```

├── services/           # طبقة الخدمات (تتعامل مع الـ API)
│   ├── api/            # الخدمات الفعلية
│   │   ├── axios.ts    # إعدادات الاتصال

├── hooks/              # React Hooks
```

**المنطق:**
- **Services** ←→ **API** (تعامل مباشر مع الـ Backend)
- **Hooks** ←→ **Services** (تستخدم الـ Services وتدي الـ State)
- **Components** ←→ **Hooks** (تستخدم الـ Hooks في الـ UI)

### الخطوة 3: إنشاء الـ Service

اعمل ملف جديد `src/services/api/commentService.ts`:

```typescript
import { apiClient, API_ENDPOINTS, buildUrl } from '../index';
import {
  ApiResponse,
  PaginatedResponse,
  Comment,
  CreateCommentRequest,
  UpdateCommentRequest,
  CommentSearchParams,
} from '../types';

class CommentService {
  // Get all comments with optional filtering
  async getComments(params?: CommentSearchParams): Promise<PaginatedResponse<Comment>> {
    const url = params ? buildUrl(API_ENDPOINTS.COMMENTS.LIST, params) : API_ENDPOINTS.COMMENTS.LIST;
    const response = await apiClient.get<PaginatedResponse<Comment>>(url);
    return response.data;
  }

  // Get single comment by ID
  async getComment(id: string): Promise<ApiResponse<Comment>> {
    const response = await apiClient.get<ApiResponse<Comment>>(API_ENDPOINTS.COMMENTS.DETAIL(id));
    return response.data;
  }

  // Create new comment
  async createComment(data: CreateCommentRequest): Promise<ApiResponse<Comment>> {
    const response = await apiClient.post<ApiResponse<Comment>>(API_ENDPOINTS.COMMENTS.CREATE, data);
    return response.data;
  }

  // Update existing comment
  async updateComment(id: string, data: UpdateCommentRequest): Promise<ApiResponse<Comment>> {
    const response = await apiClient.put<ApiResponse<Comment>>(API_ENDPOINTS.COMMENTS.UPDATE(id), data);
    return response.data;
  }

  // Delete comment
  async deleteComment(id: string): Promise<ApiResponse<void>> {
    const response = await apiClient.delete<ApiResponse<void>>(API_ENDPOINTS.COMMENTS.DELETE(id));
    return response.data;
  }

  // Get comments by post
  async getCommentsByPost(postId: string, params?: CommentSearchParams): Promise<PaginatedResponse<Comment>> {
    const url = params ? buildUrl(API_ENDPOINTS.COMMENTS.BY_POST(postId), params) : API_ENDPOINTS.COMMENTS.BY_POST(postId);
    const response = await apiClient.get<PaginatedResponse<Comment>>(url);
    return response.data;
  }

  // Get comments by author
  async getCommentsByAuthor(authorId: string, params?: CommentSearchParams): Promise<PaginatedResponse<Comment>> {
    const url = params ? buildUrl(API_ENDPOINTS.COMMENTS.BY_AUTHOR(authorId), params) : API_ENDPOINTS.COMMENTS.BY_AUTHOR(authorId);
    const response = await apiClient.get<PaginatedResponse<Comment>>(url);
    return response.data;
  }

  // Approve comment
  async approveComment(id: string): Promise<ApiResponse<Comment>> {
    const response = await apiClient.post<ApiResponse<Comment>>(API_ENDPOINTS.COMMENTS.APPROVE(id));
    return response.data;
  }

  // Reject comment
  async rejectComment(id: string): Promise<ApiResponse<Comment>> {
    const response = await apiClient.post<ApiResponse<Comment>>(API_ENDPOINTS.COMMENTS.REJECT(id));
    return response.data;
  }

  // Get replies for a comment
  async getReplies(commentId: string, params?: CommentSearchParams): Promise<PaginatedResponse<Comment>> {
    const searchParams = { ...params, parentId: commentId };
    const url = buildUrl(API_ENDPOINTS.COMMENTS.LIST, searchParams);
    const response = await apiClient.get<PaginatedResponse<Comment>>(url);
    return response.data;
  }
}

export const commentService = new CommentService();
export default commentService;
```

### الخطوة 5: إنشاء الـ Hooks

اعمل ملف جديد `src/hooks/api/useComments.ts`:

```typescript
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';
import { commentService } from '../../services';
import { 
  Comment, 
  CreateCommentRequest, 
  UpdateCommentRequest, 
  CommentSearchParams,
  ApiResponse,
  PaginatedResponse 
} from '../../services/types';

// Query keys for comments
export const commentKeys = {
  all: ['comments'] as const,
  lists: () => [...commentKeys.all, 'list'] as const,
  list: (params?: CommentSearchParams) => [...commentKeys.lists(), params] as const,
  details: () => [...commentKeys.all, 'detail'] as const,
  detail: (id: string) => [...commentKeys.details(), id] as const,
  byPost: (postId: string) => [...commentKeys.all, 'post', postId] as const,
  byAuthor: (authorId: string) => [...commentKeys.all, 'author', authorId] as const,
};

// Hook to get all comments with optional filtering
export const useComments = (params?: CommentSearchParams, enabled = true) => {
  return useQuery({
    queryKey: commentKeys.list(params),
    queryFn: () => commentService.getComments(params),
    enabled,
    staleTime: 2 * 60 * 1000, // 2 minutes
    gcTime: 5 * 60 * 1000, // 5 minutes
  });
};

// Hook to get single comment by ID
export const useComment = (id: string, enabled = true) => {
  return useQuery({
    queryKey: commentKeys.detail(id),
    queryFn: () => commentService.getComment(id),
    enabled: enabled && !!id,
    staleTime: 5 * 60 * 1000, // 5 minutes
    gcTime: 10 * 60 * 1000, // 10 minutes
  });
};

// Hook to get comments by post
export const useCommentsByPost = (postId: string, params?: CommentSearchParams, enabled = true) => {
  return useQuery({
    queryKey: [...commentKeys.byPost(postId), params],
    queryFn: () => commentService.getCommentsByPost(postId, params),
    enabled: enabled && !!postId,
    staleTime: 1 * 60 * 1000, // 1 minute
    gcTime: 3 * 60 * 1000, // 3 minutes
  });
};

// Hook to get comments by author
export const useCommentsByAuthor = (authorId: string, params?: CommentSearchParams, enabled = true) => {
  return useQuery({
    queryKey: [...commentKeys.byAuthor(authorId), params],
    queryFn: () => commentService.getCommentsByAuthor(authorId, params),
    enabled: enabled && !!authorId,
    staleTime: 2 * 60 * 1000, // 2 minutes
    gcTime: 5 * 60 * 1000, // 5 minutes
  });
};

// Hook to create new comment
export const useCreateComment = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: (data: CreateCommentRequest) => commentService.createComment(data),
    onSuccess: (data, variables) => {
      // Invalidate comments list
      queryClient.invalidateQueries({ queryKey: commentKeys.lists() });
      
      // Invalidate comments for the specific post
      queryClient.invalidateQueries({ queryKey: commentKeys.byPost(variables.postId) });
    },
    onError: (error) => {
      console.error('Comment creation failed:', error);
    },
  });
};

// Hook to update comment
export const useUpdateComment = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: ({ id, data }: { id: string; data: UpdateCommentRequest }) => 
      commentService.updateComment(id, data),
    onSuccess: (data, variables) => {
      // Update the specific comment in cache
      queryClient.setQueryData(commentKeys.detail(variables.id), data);
      
      // Invalidate list queries
      queryClient.invalidateQueries({ queryKey: commentKeys.lists() });
    },
    onError: (error) => {
      console.error('Comment update failed:', error);
    },
  });
};

// Hook to delete comment
export const useDeleteComment = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: (id: string) => commentService.deleteComment(id),
    onSuccess: (_, deletedId) => {
      // Remove the deleted comment from cache
      queryClient.removeQueries({ queryKey: commentKeys.detail(deletedId) });
      
      // Invalidate list queries
      queryClient.invalidateQueries({ queryKey: commentKeys.lists() });
    },
    onError: (error) => {
      console.error('Comment deletion failed:', error);
    },
  });
};

// Hook to approve comment
export const useApproveComment = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: (id: string) => commentService.approveComment(id),
    onSuccess: (data, variables) => {
      // Update the comment in cache
      queryClient.setQueryData(commentKeys.detail(variables.id), data);
      
      // Invalidate list queries
      queryClient.invalidateQueries({ queryKey: commentKeys.lists() });
    },
    onError: (error) => {
      console.error('Comment approval failed:', error);
    },
  });
};

// Combined hook for common comment operations
export const useCommentsManager = (params?: CommentSearchParams) => {
  const commentsQuery = useComments(params);
  const createCommentMutation = useCreateComment();
  const updateCommentMutation = useUpdateComment();
  const deleteCommentMutation = useDeleteComment();
  const approveCommentMutation = useApproveComment();

  return {
    // Queries
    comments: commentsQuery.data?.data || [],
    commentsLoading: commentsQuery.isLoading,
    commentsError: commentsQuery.error,
    pagination: commentsQuery.data?.pagination,
    
    // Mutations
    createComment: createCommentMutation.mutateAsync,
    createCommentLoading: createCommentMutation.isPending,
    createCommentError: createCommentMutation.error,
    
    updateComment: updateCommentMutation.mutateAsync,
    updateCommentLoading: updateCommentMutation.isPending,
    updateCommentError: updateCommentMutation.error,
    
    deleteComment: deleteCommentMutation.mutateAsync,
    deleteCommentLoading: deleteCommentMutation.isPending,
    deleteCommentError: deleteCommentMutation.error,
    
    approveComment: approveCommentMutation.mutateAsync,
    approveCommentLoading: approveCommentMutation.isPending,
    approveCommentError: approveCommentMutation.error,
    
    // Refetch
    refetchComments: commentsQuery.refetch,
  };
};
```

## 🎯 مثال الاستخدام في Component

```typescript
import React, { useState } from 'react';
import { useCommentsManager, useCommentsByPost } from '../hooks';

const CommentsSection: React.FC<{ postId: string }> = ({ postId }) => {
  const [newComment, setNewComment] = useState('');
  
  const {
    comments,
    commentsLoading,
    createComment,
    createCommentLoading,
    deleteComment,
  } = useCommentsManager({ postId });

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!newComment.trim()) return;

    try {
      await createComment({
        content: newComment,
        postId,
      });
      setNewComment('');
    } catch (error) {
      console.error('Failed to create comment:', error);
    }
  };

  if (commentsLoading) return <div>جاري تحميل التعليقات...</div>;

  return (
    <div>
      <h3>التعليقات</h3>
      
      {/* Add new comment form */}
      <form onSubmit={handleSubmit}>
        <textarea
          value={newComment}
          onChange={(e) => setNewComment(e.target.value)}
          placeholder="اكتب تعليقك..."
          rows={3}
        />
        <button 
          type="submit" 
          disabled={createCommentLoading || !newComment.trim()}
        >
          {createCommentLoading ? 'جاري الإضافة...' : 'إضافة تعليق'}
        </button>
      </form>

      {/* Comments list */}
      <div>
        {comments.map((comment) => (
          <div key={comment.id} style={{ margin: '10px 0', padding: '10px', border: '1px solid #ddd' }}>
            <p><strong>{comment.author.name}:</strong> {comment.content}</p>
            <small>{new Date(comment.createdAt).toLocaleString('ar')}</small>
            <button 
              onClick={() => deleteComment(comment.id)}
              style={{ marginLeft: '10px', color: 'red' }}
            >
              حذف
            </button>
          </div>
        ))}
      </div>
    </div>
  );
};

export default CommentsSection;
```

## ⚠️ أخطاء شائعة وتجنبها

### 1. نسيان تصدير الـ Service أو الـ Hooks
**المشكلة:** `Cannot find module '...'`
**الحل:** تأكد من إضافة الـ export في الملفات الرئيسية (`services/index.ts` و `hooks/index.ts`)

### 2. خطأ في الـ Types
**المشكلة:** TypeScript errors عن incompatible types
**الحل:** تأكد من إن الـ Types متطابقة بين الـ Service والـ Hooks

### 3. نسيان invalidateQueries
**المشكلة:** البيانات بتتحدثش بعد الـ mutations
**الحل:** استخدم `queryClient.invalidateQueries()` في الـ onSuccess

### 4. مشاكل في الـ Query Keys
**المشكلة:** الـ Cache بيشتغلش صح
**الحل:** استخدم query keys منظمة ومتسقة
